Employee Management System
Project Overview

The Employee Management System is a Java-based web application developed using JSP, Servlet, JDBC, and MySQL. It provides a secure admin panel where administrators can manage employee records efficiently. The system supports complete CRUD (Create, Read, Update, Delete) operations with a simple and user-friendly interface.

Features
Admin Registration
New administrators can create an account.
Stores admin details securely in the database.
Admin Login
Secure login authentication using email and password.
Only authenticated administrators can access the dashboard.
Add Employee
Add new employee details including:
Employee ID
Name
Email
Address
Salary
View Employee Details
Display all employee records in a formatted table.
Retrieve employee information directly from the database.
Update Employee Salary
Update an employee's salary using the unique Employee ID.
Displays a success message after a successful update.
Delete Employee
Remove employee records from the database using the Employee ID.
Logout
Securely ends the admin session and redirects to the login page.
